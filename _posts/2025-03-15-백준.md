## [10699번] 서울의 오늘 날짜를 출력
서울의 오늘 날짜를 "YYYY-MM-DD" 형식으로 출력한다.
<details>
  <summary>✨ <strong>코드</strong> ✨</summary>
  <div markdown="1">

```java
import java.time.LocalDate;

public class Main{
  public static void main(String[] args){
    LocalDate now = new LocalDate.now();
    System.out.print(now);
  }
}
```
